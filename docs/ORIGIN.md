# Origin

This repo exists to show that Kinetic Gain's control-plane pattern extends cleanly into biotech and diagnostics quality operations.

The primitive is not generic "lab monitoring." The real workflow pressure lives in:

- change classification decisions that need signed rationale
- stale validation evidence that blocks downstream release
- SOP acknowledgments that must land before cutover windows open
- approval and audit-trail continuity that determine whether the board can trust the packet

`gxp-change-control-board` packages that pressure into a buyer-readable and operator-readable public surface.
