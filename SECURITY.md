# Security Notes

`gxp-change-control-board` publishes synthetic biotech change-control and release-evidence data only.

It does **not** contain:

- live patient or specimen data
- production lab credentials
- proprietary assay secrets
- real manufacturing batch records
- real QA signoff packets

This repo is a public operator-surface proof for biotech and diagnostics workflows. It is not a clinical, regulatory, or safety certification claim.
